# Cloud Technology and Services

## Compute

Amazon EC2 provides virtual machines in the cloud. AWS also offers higher-level compute options such as containers and serverless services, but EC2 remains a core foundational service for understanding instance-based workloads.

## Storage

Amazon S3 is object storage designed for durability and scale. Common use cases include backups, static website hosting, media storage, and data lakes.

## Networking

Amazon VPC provides logical network isolation. Core components include subnets, route tables, internet gateways, and security groups.

## Monitoring

Amazon CloudWatch collects metrics, logs, and events from AWS resources and applications. It supports observability, alarming, and basic operational response.

## Service Selection

A core Cloud Practitioner skill is identifying which service category fits a problem:

- Compute for processing workloads
- Storage for durable data retention
- Networking for connectivity and isolation
- Monitoring for operational visibility
