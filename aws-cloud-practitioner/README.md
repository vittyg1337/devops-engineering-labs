# AWS Cloud Practitioner

## Purpose

This section contains the AWS foundation phase of the repository. It is structured to build practical familiarity with core AWS services while reinforcing the concepts required for entry-level cloud and DevOps work.

## Certification Target

The learning path is aligned to the AWS Certified Cloud Practitioner certification. The emphasis is on understanding core AWS services, security fundamentals, pricing concepts, and common cloud use cases rather than deep implementation complexity.

## Lab-Based Learning Approach

Each topic is handled as an individual lab with a clear objective, scoped AWS services, execution steps, and a short summary of lessons learned. This keeps the track practical, reviewable, and easy to expand as the repository grows into more advanced DevOps domains.
