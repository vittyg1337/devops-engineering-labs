# AWS Cloud Practitioner Labs

This directory contains hands-on labs for the AWS Cloud Practitioner track. Each lab focuses on one core service area and ties the activity back to an exam domain and a practical cloud concept.

Labs in this track:

- Lab 01: IAM users and groups
- Lab 02: S3 static website hosting
- Lab 03: EC2 Linux basics
- Lab 04: CloudWatch monitoring
- Lab 05: VPC basics
- Lab 06: Cost estimation
