# Lab 06 - Cost Estimation

## Objective

Practice estimating AWS costs for a simple workload and review the tools and concepts used to understand pricing and billing.

## Services Used

- AWS Pricing Calculator
- AWS Billing and Cost Management
- AWS Free Tier

## What I Built

- Estimated the cost of a small AWS workload
- Reviewed how service pricing varies by usage model
- Identified the role of billing tools and cost visibility features

## Key Concepts

AWS pricing is based on consumption. Cost depends on the service, region, configuration, and amount of usage.

The AWS Pricing Calculator helps forecast expected spend before deployment. Billing and Cost Management provides visibility into actual charges after resources are used.

## Steps Performed

1. Select a sample workload to estimate.
2. Enter service assumptions into the AWS Pricing Calculator.
3. Review projected monthly cost.
4. Compare the estimate with Free Tier considerations.
5. Summarize the main pricing drivers.
