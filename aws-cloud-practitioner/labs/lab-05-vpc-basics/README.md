## Key Concepts

An Amazon VPC is a logically isolated virtual network in AWS. It allows workloads to be placed into controlled IP ranges and network segments.

Subnets divide the VPC into smaller network sections. Route tables determine where traffic is sent, and an internet gateway provides a path for internet-connected resources.
